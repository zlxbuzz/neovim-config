# neovim-config
### neovim configuration
mv neovim-config nvim
部分字符无法展示，需要下载对应的符号字体库，然后在对应的终端选择字体（比如Hack Nerd Font）
```sh
git clone https://github.com/ryanoasis/nerd-fonts.git --depth 1
cd nerd-fonts
bash ./install.sh
```
